Survive. Explore. Escape.

Dungeon Run is a dark 2D roguelike game concept where every run takes places inside a dangerous dungeon filled with enemies, treasure, traps, and randomly generated rooms.

Explore the Game

Core Game Features

Random Dungeons

Each run creates a new dungeon layout with different rooms, paths, enemies, and treasure locations.

Enemy AI

Enemies chase, attack, and react to the player using simple movement logic and combat behavior.

Weapons & Loot

Players can collect swords, potions, armor, keys, and rare items that change how each run feels.

Boss Rooms

At the end of each dungeon level, players face a stronger Boss enemy with unique attacks.

How The Game Works

1. Enter the Dungeon

The player starts with basic health, a simple weapon, and one goal: survive.

2. Explore Rooms

Each room can contain enemies, loot, traps, hidden paths, or locked doors.

3. Upgrade Your Build

Collect items that improve speed, damage, defense, or special abilities.

4. Defeat the Boss

Clear the dungeon floor by defeating a boss and moving deeper into the dungeon

Computer Science Behind It

Procedural Generation

Algorithms create different dungeon layouts instead of manually designing every room.

Collision Detection

The game checks when the player touches walls, enemies, attacks, items, or exits.

State Machines

Enemy behavior changes between idle, chasing, attacking, and defeated states.
